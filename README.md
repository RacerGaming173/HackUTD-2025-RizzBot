# RizzBot
---
An AI-powered in-ear conversational coach to spice up your conversations and create new bonds. 
# Features
- **Near real-time in-ear feedback:** RizzBot provides dynamic conversation suggestions using generative AI based on the other party's responses.
- **Seamless speech-to-text and text-to-speech transcription**: RizzBot is capable of interpreting sentences and ignoring noise in a variety of environments.
- **Bluetooth connectivity:** Conversation suggestions are communicated to the user through a Bluetooth headset.
- **Conversation status indicator:** LEDs indicate the quality of the conversation on a scale of Bad, OK, and Good.
# Setup Instructions
1. Install dependencies
2. Run `python src/main.py`
